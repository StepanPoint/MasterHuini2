# MasterHuini 🎵

**AI Auto-Mix платформа для DJ и музыкантов**

Профессиональный веб-сервис для автоматического сведения, мастеринга и микширования треков с использованием искусственного интеллекта.

## 🚀 Развёртывание на GitHub Pages

1. Создай новый репозиторий на GitHub
2. Загрузи все файлы из этого архива в корень репозитория
3. Перейди в **Settings → Pages**
4. В разделе **Source** выбери ветку `main` и папку `/ (root)`
5. Нажми **Save**
6. Через 1–2 минуты сайт будет доступен по адресу:
   `https://ТВО-ИМЯGITHUB.github.io/ИМЯ-РЕПОЗИТОРИЯ`

## 📁 Структура файлов

```
masterhuini/
├── index.html          # Основной файл (SPA)
├── css/
│   └── style.css       # Все стили
├── js/
```

## 🎛 Страницы сайта

| Страница | Описание |
|----------|----------|
| **Главная** | Hero с inline-плеером, статистика, возможности, список треков |
